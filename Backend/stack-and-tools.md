A brief overview of the crates and other tools being used in this project.

| Layer           | Technology                 | Rationale                                    |
| --------------- | -------------------------- | -------------------------------------------- |
| Language        | Rust (Tokio async runtime) | Safety and concurrency                       |
| ORM             | SeaORM                     | Type-safe async ORM                          |
| Database        | PostgreSQL via Supabase    | Reliable managed hosting                     |
| Config          | dotenvy                    | Secure environment variable management       |
| Docs            | Obsidian                   | Internal documentation and architecture maps |
| Version Control | Git + GitHub               | Team collaboration                           |
